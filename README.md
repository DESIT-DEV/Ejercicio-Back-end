## Instrucciones


**Ejercicio práctico** **(Back-end)**

Una empresa de alimentos por motivo de fin de año desea hacer una donación de sus productos a las personas que más lo necesitan para ello ha decido crear canastas navideñas.

Para ello ha tomado en cuenta el poder desarrollar un sistema para llevar el control de las personas que están siendo beneficiadas con esta ayuda y personas que le ayuden a entregar dichas canastas, por ende debe considerarse estos puntos:

1. El sistema debe ser accedido únicamente por las personas que estarán encargados en entregar las canastas y el administrador.

2. Se debe garantizar que cada canasta se entre únicamente a una persona (no entregar dos canastas a la misma persona) y debe ser mayor de edad.

3. Saber la cantidad de canastas que entrego cada encargado.

4. Llevar el registro de la fecha que se entregó cada canasta.

5. Tener la posibilidad de editar la información o nombre del beneficiado (por cualquier detalle de ortografía) y además poder eliminarlo al no percatarse de haberlo agregado más de una vez (ya que solo se debe entregar una canasta por persona).

6. Máximo  de canasta a entregar serán 10 por total (evitar que se entregan más).

7.  Saber cuántos hombres y cuantas mujeres recibieron canastas.

## Criterios a evaluar

1. Utilizar el lenguaje de programación JavaScript con el framework Node js o utilizar php como lenguaje de programación

2. Manejo de buenas prácticas y control de errores en estados de peticiones al servidor

3. Flujo y Base de datos utilizado en MySQL

4. Archivos de programación que se utilizaron (y porque se utilizó ese lenguaje).

5. Funcionalidad del ejercicio práctico.

6.  Cumplimento de los puntos a considerar.
